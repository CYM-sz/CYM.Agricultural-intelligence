# CYM Agricultural Intelligence

This repository provides a plant knowledge database and integrates with GPT for intelligent plant care recommendations.

## Project Structure
- `plant_knowledge/`: Contains the plant knowledge database.
- `gpt_integration/`: Logic to interact with the GPT model.

## Setup Instructions
1. Clone the repository.
2. Install required dependencies.
3. Run the application.
